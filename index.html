My HTML Cheat Sheet

Basic Tags

<!DOCTYPE html>
<html lang="en"></html>
<head></head>
<title>Page Title</title>

Attributes

<a href="url" target="_blank">link text</a>
<img src="img.jpg" alt="description" width="500" height="600">

Useful Snippets

<iframe width="560" height="315" src="https://www.youtube.com/embed/videoID" frameborder="0" allowfullscreen></iframe>

Additional Notes

Always validate your HTML using the W3C validator. Remember to add comments for clarity: <!-- This is a comment -->. Study semantic HTML for better accessibility and SEO.
