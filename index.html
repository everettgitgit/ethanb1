My HTML Cheat Sheet

Basic Tags

<!DOCTYPE html>
<html lang="en"></html>
<head></head>
<title>Page Title</title>

Text Tags and Formatting

Basic text formatting tags to structure and style text content.

<h1>Heading 1</h1>
<h2>Heading 2</h2>
<p>Paragraph</p>
<strong>Bold Text</strong>
<em>Italic Text</em>

Lists

Creating ordered and unordered lists, as well as description lists.

<ul>
<li>List Item 1</li>
<li>List Item 2</li>
</ul>
<ol>
</ol>
<dl>
</dl>

Graphical Elements

Adding images and figures with captions to your HTML content.

<img src="image.jpg" alt="Image Description">
<figure>
<img src="image.jpg" alt="Image Description">
<figcaption>Caption for the image.</figcaption>
</figure>

Forms

Creating forms for user input.

Opening tag for a form

<form action="/submit">

Input field for user text input

<input type="text" name="username">

Input field for password input

<input type="password" name="password">

Submit button for the form

<input type="submit" value="Submit">
</form>

Tables

Creating tables for data representation.

<table>
<tbody>
<tr>

content "Row 1, Cell 1"

<td>Row 1, Cell 1</td>

content "Row 1, Cell 2"

<td>Row 1, Cell 2</td>
</tr>
</tbody>
</table>

Attributes

anchor tag with href attribute

<a href="https://example.com">Click here</a>

image tag with src and alt attributes

<img src="image.jpg" alt="Description">

anchor tag with href, target, and title attributes

<a href="url" target="_blank" title="Tooltip">Link</a>

image tag with src, alt, width, and height attributes

<img src="image.jpg" alt="Image Description" width="500" height="300">

input tag with type, disabled, and placeholder attributes

<input type="text" disabled placeholder="Disabled Input">

a div tag with class attribute

<div class="classname">...</div>

a span tag with inline style attribute

<span style="color: red;">Colored Text</span>
